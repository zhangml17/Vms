# Zms
## Use
```
npm install
npm run serve
```
## Prepare
### Install
```
npm i mxgraph --save
npm i exports-loader --save
npm i script-loader --save  
npm i node-sass sass-loader --save 
```
### Config
+ add vue.config.js
+ add utils/index.js as global object

## Operation
+ 默认ctrl + 鼠标左键可以进行多个cell的选择
